#  Movie Data Analytics & Sakila Queries 

## Descripción
[cite_start]Este proyecto corresponde a la **Evaluación Final del Módulo 2**[cite: 3]. [cite_start]Su objetivo principal es poner en práctica habilidades de análisis de datos estructurado en dos bloques: en primer lugar, la extracción automatizada de datos cinematográficos desde una API externa para su posterior almacenamiento y análisis en una base de datos relacional [cite: 8, 10][cite_start]; y en segundo lugar, la resolución de consultas de negocio complejas sobre la base de datos de ejemplo *Sakila*[cite: 42, 46].
